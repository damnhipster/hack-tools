Hosting

When you present your hack, you don't want to fiddle with cables and laptops.

Hosting should be sorted out as soon as you start.

But make sure you're not uploading anything sensitive!.

Github

If you have a simple web page (index.html and no back end), upload your project to Github.

You will need to:

  1. Make sure your project is using Git before using Github
  2. Create/use a github account (e.g. hems-account)
  3. Create a repo and give it a name (e.g. my-hack)
  4. Create and push/upload a branch called gh-pages
  5. You're page is on hems-account.github.io/my-hack

Github.com

ngrok

You can run your hack on your machine, but expose it online.

For Macs:

Install HomeBrew 


ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Install ngrok 


brew install ngrok

Start ngrok on the port your app is running (e.g. 9001) 


ngrok http 9001
